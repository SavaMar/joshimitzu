# Joshimitsu BJJ Website

Modern website for Joshimitsu BJJ — a Brazilian Jiu-Jitsu school in Biberist, Switzerland.

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **TypeScript**
- **Supabase** (planned — admin panel placeholder ready)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page with all sections |
| `/what-is-bjj` | About Brazilian Jiu Jitsu |
| `/trial-training` | Free trial booking info |
| `/blog` | Blog listing |
| `/blog/[slug]` | Individual blog articles |
| `/events` | Upcoming events |
| `/about` | About the school |
| `/contact` | Contact information |
| `/data-protection` | Privacy policy |
| `/admin` | Admin panel (Supabase TBD) |

## Theme

Bright theme with white, black, and blue (`#38bdf8`) accent colors.
